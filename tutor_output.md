# Understanding Multidigit Multiplication

Multidigit multiplication can be challenging at first, but once we break it down into smaller steps, it becomes much more manageable. Let’s learn how to multiply a two-digit number by another two-digit number.

## Problem Example: Multiply 23 by 45

### Step-by-step Explanation:

1. **Set Up the Problem**: 
   - Write the numbers one under the other, aligning them by place value. 
   - For example:
   ```
     23
   x 45
   ------
   ```

2. **Multiply the Ones Place First**:
   - Multiply the bottom number's ones place (5) by the top number (23).
   - 5 times 3 (the ones place of 23) equals 15. Write down 5 and carry over 1.
     ```
     5 (carry 1) 
     ```
   - Next, multiply 5 by 2 (the tens place of 23), which equals 10. Add the carry-over (1) to get 11. Write it down to the left.
   - So, the first row of our answer looks like this:
     ```
     115
     ```

3. **Multiply the Tens Place Next**:
   - Now, multiply the bottom number's tens place (4) by the top number (23).
   - Since this is the tens place, we will add a zero at the end of our answer for this row (effectively making it worth 40).
   - We start by multiplying:
     - 4 times 3 equals 12. Write down 2 and carry over 1.
     ```
     12 (carry 1)
     ```
   - Then, multiply 4 by 2 equals 8. Add the carry-over (1) to get 9.
   - This gives us the second row:
     ```
     920
     ```

4. **Add the Two Results Together**:
   - Now we will add our two partial products together:
   ```
      115
   + 920
   ------
    1035
   ```
   - Therefore, 23 times 45 equals **1035**.

### Conclusion:
To multiply multidigit numbers:
- Multiply the ones place first.
- Then, multiply the tens place (remembering to add a zero).
- Finally, add the two results together.
